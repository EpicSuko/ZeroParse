Architecture Overview

  • ZeroParse - Main API facade providing static parse() methods
  • JsonParser - Core parser implementing zero-copy parsing
  • StackTokenizer - Stack-based tokenizer for building AST
  • AstStore - Storage for the Abstract Syntax Tree
  • Node* classes - Node-based views over the AST (used by parser)
  • Json* classes - Immutable JSON value types (used by API consumers)
  • InputCursor - Abstract interface for reading from various sources
  • ZeroParseRuntime - Manages object pools via FastPool
  • Utf8Slice - Zero-copy UTF-8 string representation
